=head1 SYNOPSIS

=head1 DESCRIPTION

Thos module does not implement an actual soap server daemon, but the
needs to create the server side.  The server daemon is implemented
by L<XML::Compile::SOAP::Daemon|XML::Compile::SOAP::Daemon>.
